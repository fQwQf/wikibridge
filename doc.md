# 三个端
运行我们软件的三种物理机:

1. 服务器:frps + 管理面板(frp的连接端口管理 + 用户系统)
2. 用户端:GUI软件(知识库server + 魔改过的frpc)
3. 用户的用户:知识库client + chatbox client(用开源的)
【知识库的server client】 和 【frp管理面板】相互独立,但是要足够定制化
